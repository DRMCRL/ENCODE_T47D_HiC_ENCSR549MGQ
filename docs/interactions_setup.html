Introduction

The output from MaxHiC is extremely large and for some bin-sizes, near impossible to wrangle on a standard laptop. In this workflow, each set of outputs containing the cis interactions will be loaded and reduced to only the significant interactions. These far smaller sets of interactions are easily managed in a standard compute environment. Analyses were run for genomic bins of 10, 20, 40, 100 and _500_kb.

allGR <- read_rds(
  here::here("output/allGR.rds")
)
sq <- seqinfo(allGR)

Data Preparation

sz <- 20000

20kb Bins

bins <- import.bed(
  glue("data/hic/hic_results/matrix/merged/raw/{sz}/merged_{sz}_abs.bed"),
  seqinfo = sq
)
cis_int <- vroom(
  glue("output/MaxHiC/merged/{sz}/cis_interactions.txt.gz"),
  col_types = "dddd-d--"
)

By default, any bin pairs with no observed interactions were excluded fro the Max-HiC output. Using 20kb bins, a total of 40,041,031 non-zero interactions were observed between bin pairs. The median observed interaction count was 1, whilst the maximum was 181.

By default, the FDR provided by Max-HiC is incorrect and was ignored. The MaxHiC output was defined as a GInteractions object, with correct FDR added and interactions filtered for those with an FDR < 0.05

gi <- GInteractions(
  anchor1 = bins[as.integer(cis_int$bin1ID)],
  anchor2 = bins[as.integer(cis_int$bin2ID)],
  counts = as.integer(cis_int$observed_interactions),
  exp_interactions = cis_int$exp_interactions,
  p = exp(-cis_int$neg_ln_p_val)
)
seqinfo(gi) <- sq
gi$fdr <- p.adjust(gi$p, "fdr")
gi$distance <- pairdist(gi)
gi$logObsExp <- log2(gi$counts / gi$exp_interactions)
gi <- subset(gi, fdr < 0.05)
write_rds(
  gi,
  here::here(
    "output",
    glue("gi_{sz}.rds")
  ),
  compress = "gz"
)
gc()

After filtering interactions to only those considered significant:

  • 8,469 bin-pair interactions were retained.
  • The observed interactions between bins ranged between 6 and 181
  • The log2 ratio of of observed to expected counts ranged between 1.7 and 4.23. These values represent enrichment for interactions well above that expected by the background model.
gi %>%
  anchors("first") %>%
  seqnames() %>%
  table() %>%
  enframe(
    name = "chr",
    value = "count"
  ) %>%
  mutate(
    chr =factor(chr, levels = seqlevels(sq)),
    count = as.integer(count)
  ) %>%
  ggplot(
    aes(chr, count)
  ) +
  geom_col() +
  labs(
    x = "Chromosome",
    y = glue("Total Significant Cis Interactions ({sz/1e3}kb bins)")
  ) +
  scale_y_continuous(expand = expansion(c(0, 0.05)))
