Introduction

The output from MaxHiC is extremely large and for some bin-sizes, near impossible to wrangle on a standard laptop. In this workflow, each set of outputs containing the cis interactions will be loaded and reduced to only the significant interactions. These far smaller sets of interactions are easily managed in a standard compute environment. Analyses were run for genomic bins of 10, 20, 40, 100 and _500_kb.

Data Preparation

10kb Bins

sz <- 10000
bins <- import.bed(
  glue("data/hic/hic_results/matrix/merged/raw/{sz}/merged_{sz}_abs.bed"),
  seqinfo = sq
)
cis_int <- vroom(
  glue("output/MaxHiC/merged/{sz}/cis_interactions.txt.gz"),
  col_types = "dddd-d--"
)
fdr <- p.adjust(exp(-cis_int$neg_ln_p_val), "fdr")
keepInt <- fdr < 0.05

By default, any bin pairs with no observed interactions were excluded from the Max-HiC output. Using 10kb bins, a total of 51,692,192 non-zero interactions were observed between bin pairs. The median observed interaction count was 1, whilst the maximum was 124.

By default, the FDR provided by Max-HiC is incorrect and was ignored. The MaxHiC output was defined as a GInteractions object, with correct FDR added and interactions filtered for those with an FDR < 0.05.

gc()
gi <- GInteractions(
  anchor1 = bins[as.integer(cis_int$bin1ID[keepInt])],
  anchor2 = bins[as.integer(cis_int$bin2ID[keepInt])],
  counts = as.integer(cis_int$observed_interactions[keepInt]),
  exp_interactions = cis_int$exp_interactions[keepInt],
  p = exp(-cis_int$neg_ln_p_val)[keepInt],
  fdr = fdr[keepInt]
)
gi$distance <- pairdist(gi)
gi$logObsExp <- log2(gi$counts / gi$exp_interactions)
seqinfo(gi) <- sq
write_rds(
  gi,
  here::here(
    "output",
    glue("gi_{sz}.rds")
  ),
  compress = "gz"
)
gc()

After filtering interactions to only those considered significant:

  • 2,942 bin-pair interactions were retained.
  • The observed interactions between bins ranged between 6 and 124
  • The log2 ratio of of observed to expected counts ranged between 2.02 and 4.41. These values represent enrichment for interactions well above that expected by the background model.
  • The median interaction distance was 110kb
  • The longest interaction was between bins separated by 191Mb
